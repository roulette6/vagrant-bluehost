# Using Sequel Pro

Connect via SSH

- Name       : <project name>
- MySQL Host : 127.0.0.1
- Username   : root
- Password   : root
- Database   : <optional>
- Port       : leave blank
- SSH Host   : 127.0.0.1
- SSH User   : vagrant
SSH Password : Either "vagrant" or look up the SSH key stored in .vagrant/insecure_private_key
- SSH Port   : 2222
